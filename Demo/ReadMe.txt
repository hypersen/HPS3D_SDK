HPS3D_SDK java平台使用注意事项：
在java平台上使用HPS3D_SDK,我司将以 libhps3d32 或 libhps3d64 形式命名的jar包以及libhps3dxx.dll动态库发布；
使用方法：将libhps3dxx.jar拷贝至项目目录下(如：../lib/)；右键选择该文件--->Build Path 将该文件添加到工程库路径中
如提示找不到xxx.dll文件，则需手动拷贝libhps3dxx.dll到项目bin目录下，重新编译即可;
具体使用方法请参考相应的demo程序；
